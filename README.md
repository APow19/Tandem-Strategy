# Tandem Strategy

Static marketing site for Tandem Strategy, combining CGMA-qualified financial management expertise with specialist purchasing and international sourcing knowledge. Based in Barnsley & Wakefield, serving South & West Yorkshire and beyond — 40 years of combined experience.

## Tech stack

- Pure HTML, CSS, and JavaScript — no build tools or framework
- Opens directly in a browser — no local server needed

## Files

- `index.html` — the main single-page site
- `templates.html` — free business templates resource page
- `css/styles.css` — main site styles
- `css/templates.css` — styles for the templates page
- `js/main.js` — main site interactivity
- `js/templates.js` — templates page interactivity
- `templates/` — free downloadable business templates (business plan, client proposal, employment contract, invoice, NDA, quote/estimate, scope of work, terms & conditions)
